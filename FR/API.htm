Interface de programmation

Interface de programmation

Objet, Attribue et fonction auquel on peut accéder.

Ressource

·         Type Metal

·         Type Carbon

·         Type Loicium

RessourceSet

·         constructeur (quantité Metal, Carbon, Loicium) -> RessourceSet

·         constructeur () -> RessourceSet

·         constructeur (RessourceSet) -> RessourceSet

·         méthode at(Type de Ressource) -> Quantité de ressource

Coord

·         constructeur (x, y, z) -> Coord

·         attribut X -> position en X

·         attribut Y -> position en Y

·         attribut Z -> position en Z

Building

·         Type CommandCenter

·         Type MetalMine

·         Type CarbonMine

·         Type LoiciumFilter

·         Type Factory

·         Type Laboratory

·         Type CarbonicCentral

·         Type SolarCentral

·         Type GeothermicCentral

BuildingTab

·         méthode size() -> nombre de type de building existant

·         méthode at(type de Building) -> quantité de buildings de ce type.

Cannon

·         Type Cannon1

·         Type Cannon2

·         Type Cannon3

·         Type Cannon4

·         Type Cannon5

·         Type Cannon6

CannonTab

·         méthode size() -> nombre de cannons existant

·         méthode at(type de Cannon) -> quantité de cannons de ce type

Planet

·         méthode isFree() -> Oui ou non (true ou false)

·         attribut coord -> Position de la planète, de type Coord

·         attribut playerId -> identifiant du joueur qui la possède

·         attribut buildingList-> liste de Building (type BuildingTab)

·         attribut cannonTab -> liste des Cannon (type CannonTab)

·         attribut ressourceSet -> liste des ressources, de type RessourceSet

Ship

·         Type Mosquito

·         Type Hornet

·         Type Vulture

·         Type Dragon

·         Type Behemoth

·         Type Azathoth

·         Type Queen

·         Type Cargo

·         Type LargeCargo

ShipTab

·         méthode size() -> nombre de type de vaisseaux

·         méthode at(Type de Ship) -> quantité de vaisseaux de ce type

Fleet

·         attribut id -> identifiant de la flotte

·         attribut playerId -> identifiant du joueur qui la possède

·         attribut coord -> Coordonnées actuel de la flotte (type Coord)

·         attribut origine -> Coordonnées de la planète d'origine (type Coord)

·         attribut shipList -> Liste des vaisseaux dans la flotte (type ShipTab)

·         attribut ressourceSet -> liste des ressources, de type RessourceSet

PlanetAction

·         Type Building

·         Type Ship

·         Type Cannon

·         attribut action -> identifiant de ce type d'action

·         attribut building -> type de building associé, si applicable.

FleetList

·         méthode at(indice de la flotte voulue) -> Flotte a l'indice demandé (type Fleet)

·         méthode range() -> permet d'utilise l'instruction for sur la liste

FleetAction

·         Type Nothing

·         Type Move

·         Type Harvest

·         Type Colonize

·         Type Drop

·         constructeur (type de FleetAction, Coordonnée optionnelle)

·         attribut action -> type d'action

·         attribut target -> Destination, si applicable

Fonctions libres

fonction directionRandom() -> fournit un direction définit aléatoirement

fonction directionFromTo(Coordonnée d'origine, destination) -> direction (type Coord)

fonction makeBuilding(type de Building) -> Requête de fabrication de building. (PlanetAction)

fonction makeShip(Type de Ship) -> Requête de fabrication de vaiseau. (PlanetAction)

fonction makeCannon(Type de Cannon) -> Requête de fabrication de canon. (PlanetAction)
